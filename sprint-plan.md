# خطة السبرنتات - منصة محاسبتك (Frontend Build Plan)

## تعليمات عامة لـ AI Agent (اقرأها قبل أي سبرنت)

1. في أول كل سبرنت: افتح ملف `PROGRESS.md` (لو مش موجود، أنشئه) واقرأ آخر حالة مسجلة فيه قبل ما تبدأ أي كود.
2. لما تخلص السبرنت: **حدّث `PROGRESS.md` إجباريًا** بالفورمات المحدد تحت في قسم "فورمات ملف PROGRESS.md"، قبل ما تعتبر السبرنت خلص.
3. متنقلش لسبرنت جديد إلا لو الـ Definition of Done بتاع السبرنت الحالي كله علامة صح.
4. لو لقيت نفسك محتاج تكسر ترتيب السبرنتات (مثلاً تحتاج حاجة من سبرنت لاحق)، سجل السبب في `PROGRESS.md` تحت "ملاحظات وانحرافات" وكمل بحذر.
5. كل الكود الجديد يتبني على الـ tokens والـ shell اللي اتعملوا في سبرنت 1 و2 — ممنوع إعادة تعريف ألوان أو خطوط جديدة في صفحة لوحدها.

## Sprint 1 — Design Tokens & Base Styles
**الهدف:** كل الألوان/الخطوط/المسافات في مكان واحد.

- [x] `css/base.css`: CSS variables للألوان (البنفسجي الأساسي، الداكن بتاع السايدبار، الأبيض، الرمادي للنصوص الثانوية، ألوان الحالة success/warning)
- [x] تعريف الخط (Cairo أو Tajawal) عبر Google Fonts أو self-hosted، وتطبيقه كـ default
- [x] Typography scale (عناوين، نص أساسي، نص صغير/captions)
- [x] Spacing scale موحّد (4px/8px steps) كـ variables
- [x] Border-radius واحد موحّد للكروت والأزرار
- [x] تعريف الـ breakpoints في تعليق أعلى الملف (مثلاً: mobile <768px, tablet 768-1024px, desktop >1024px)
- [x] صفحة اختبار بسيطة (`test-tokens.html`) تعرض كل الألوان والخطوط جنب بعض للمراجعة

**Definition of Done:** تقدر تفتح `test-tokens.html` وتشوف كل الـ design system من غير ما تكتب أي CSS جديد.

## Sprint 2 — الـ Shell (Navbar + Sidebar) — Mobile First
**الهدف:** الهيكل العام اللي كل صفحة هتستخدمه، شغال على أصغر شاشة الأول.

- [x] بناء الـ Header (لوجو/بروفايل، بحث، أيقونات إشعارات) على مقاس 375px
- [x] بناء الـ Bottom Navigation (للموبايل) بنفس عناصر السايدبار
- [x] بناء الـ Sidebar (للديسكتوب فقط، يظهر من `768px` أو `1024px` فما فوق)
- [x] الـ Header والـ Nav يتحولوا لملف component منفصل (`components/navbar.js`, `components/sidebar.js`) قابلين للتضمين في أي صفحة
- [x] اختبار الـ Shell على 3 مقاسات: 375px, 768px, 1440px

**Definition of Done:** الـ Shell شغال ومتماسك على الموبايل والديسكتوب من غير أي محتوى صفحات جواه لسه.
